## Architecture

**Next.js 15 App Router + Prismic CMS + Slice Machine.**

Content is managed in Prismic (repo: `atrancasv2`). Pages are `page` documents composed of slices chosen in Prismic's editor. The site has two routes:

- `src/app/page.tsx` â†’ renders the `home` UID document
- `src/app/[uid]/page.tsx` â†’ renders any other `page` document by UID

Both fetch from Prismic via `createClient()` (`src/prismicio.ts`) and render with `<SliceZone>` using the slice registry in `src/slices/index.ts`.

### Slices (`src/slices/`)

Each slice is a directory with `index.tsx` (React component), `model.json` (field schema), and `mocks.json`. Slices receive a `slice` prop typed via `SliceComponentProps<Content.XxxSlice>` from `@prismicio/react`.

**`src/slices/index.ts` is auto-generated by Slice Machine â€” never edit it manually.** Create or modify slices through the Slice Machine UI at `http://localhost:9999`, then edit the generated `index.tsx`.

Slices with variants (e.g. `Categories`) put variant components in a `variants/` subdirectory and select them from the main `index.tsx`.

### Components (`src/components/`)

Shared UI: `Bounded` (layout wrapper with padding props), `Heading`, `PrismicRichText`, `PrismicSerializer` (shared `commonComponents` JSX map for rich text), `EmblaCarousel`, `Header`, `Footer`, `Analytics`.

### i18n

next-intl is wired up but **currently disabled** via `integrations.config.tsx` (`isI18nEnabled: false`). Locale is read from a `locale` cookie. Dictionaries live in `dictionary/en.json` and `dictionary/es.json`. The app defaults to Spanish (`lang="es"` on `<html>`).

### Analytics

`src/components/Analytics.tsx` injects GTM (`GTM-5QX7JFRZ`), GA4 (`G-QWT8LWXJ5J`), and Google Ads (`AW-808610482`) scripts. IDs are hardcoded there. The Contact slice fires a Google Ads conversion event on successful form submission.

### Contact form

`src/slices/Contact/index.tsx` uses react-hook-form + yup + EmailJS. EmailJS credentials (`SERVICE`, `TEMPLATE`, user ID) are hardcoded in that file. Phone validation expects exactly 9 digits (Spanish format).

### Styling

Tailwind CSS v4. Brand accent color: `#c39f77`. Custom heading sizes defined in `tailwind.config.js`. Use `clsx`/`tailwind-merge` for conditional class merging.

### Prismic preview & revalidation

- `src/app/api/preview/route.ts` â€” enters draft preview mode
- `src/app/api/exit-preview/route.ts` â€” exits preview
- `src/app/api/revalidate/route.ts` â€” ISR revalidation webhook (tag: `"prismic"`)

In production, Prismic content is `force-cache`; in dev it revalidates every 5 seconds.
